## Prerequisites

- npm and gulp
- Download [NodeJs](https://nodejs.org/en/download/).
### Development
- Install ExpressJS
<code>npm install express</code>
- Install Nodemon (watch file changes): 
<code>npm install nodemon --save-dev</code>
- Stop: Ctrl + C 
- Install Morgan (middleware) watch request from client to server: 
<code>npm install morgan --save-dev</code>
- Install handlebars: 
<code>npm install express-handlebars --save-dev</code>
## Run
The first run
- Step 1:
<code>npm i</code>
- Step 2: 
<code>npm run compiler</code>
- Step 3: 
<code>npm start</code>

Next runs:
<code>npm start</code>
