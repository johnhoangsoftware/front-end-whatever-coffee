## Prerequisites

- npm and gulp
- Download [NodeJs](https://nodejs.org/en/download/).
### Development
- Install ExpressJS
+ npm install express
- Install Nodemon
watch file changes
Stop: Ctrl + C 
+ npm install nodemon --save-dev
- Install Morgan (middleware)
watch request from client to server 
+ npm install morgan --save-dev
-Install handlebars
+ npm install express-handlebars --save-dev
## Run
Terminal: npm start
